Ultra Pivot

Less code | Faster Development | Ready To Use Themes

What it is ?

UltraPivot is an extension over Github's open source project PivotTable with more cool feature and and better performance with rich developer friendly API for customization.

LEARN MORE TRY IT YOURSELF

Less Coding

UltraPivot is an easy table builder - just provide the data and it will take care of creating tablesfor you, add content and style it to look the way you like.

Mobile Friendly

Teested and works well with most of the mobile browser makes it more mobile-friendly. See the complete list of suppoted browser here.

Stylish Themes

UltraPivot offers many themes "out-of-the-box" which you can download and to make your table UI rich and flexible. None the less you can create your own themes too and contribute back if you like to.

How To Use ?

No matter you are experienced or new to javaScript, this is going to be a piece of cake for you

1

Download

Download the .zip or tar.gz file from the latest release on github or get the development copy to try latest features

2

Add To Your Project

Add the ultra-pivot.js and ultra-pivot.css files to your web project.

3

Ready To Use

And that's it. You are ready to use it. Follow documentation to explore all the features now.

Cross Browser Support

Ultra Pivot is built to support all modern desktop and mobile web browsers
